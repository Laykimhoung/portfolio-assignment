02. The Origin

My Journey

How I went from a different path to discovering a passion for code.

The Beginning

Unexpected Path

I never planned to study Computer Science. I originally considered becoming a doctor because many of my relatives work in medicine.

High School

Discovering Tech

During my final year, I explored many different majors. I became interested in technology after researching IT and programming.

The Choice

Building Software

I chose Computer Science because I wanted to build software rather than work with networking or hardware. I knew nothing about programming at the beginning, not even what an IDE was or how websites worked.

Turning Point

Hello World

The first time I successfully ran a webpage using Live Server became a turning point that motivated me to continue learning.

Collaboration

Embracing GitHub

Discovering GitHub completely changed how I learned software development and teamwork. I even introduced it into a university team project before it was taught in class.
